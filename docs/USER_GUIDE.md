# WAVE SS+ User Guide

How to use the SS PLUS Management System.

## Navigation

- **Home**: Dashboard with quick access to documents, records, and audits.
- **Documents**: Browse the 7 company documents. Use sidebar and "On this page" TOC to jump to sections.
- **Records**: Browse 50+ management forms. Filter by department (Finance, HR, Sales, Operations, Inventory, Administrative, Quality) and search.
- **Audits**: View audit reports and detailed findings.

## Documents

- Open a document. Use **Export** in the header:
  - **Print Document**: Opens print dialog.
  - **Download PDF**: Generates A4 PDF from full MD when available (cover, sections, headers/footers).

## Records

- Open a record from Records Repository. Use **Print** or **Download PDF** to export.

## Audits

- Open an audit. Use **Download Full PDF Report** to export as PDF.

## Language and Theme

- Use the language switcher for EN/FR/AR. Arabic uses RTL.
- Use the theme toggle for light/dark mode.

## Search

- Use global search (e.g. Cmd+K or header search) to find documents and content.
